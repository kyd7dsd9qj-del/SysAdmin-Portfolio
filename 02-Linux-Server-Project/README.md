# Проект 2: Развертывание серверной инфраструктуры на Linux

## Цель
Настроить полный стек серверных служб для внутренней IT-инфраструктуры компании.

## Задачи
- Настройка DNS-сервера (dnsmasq) с доменом practicumsuperstore.ru
- Развертывание NTP-сервера (Chrony) для синхронизации времени
- Установка и настройка почтового сервера Postfix
- Настройка веб-сервера Nginx с базовой аутентификацией (htpasswd)
- Настройка FTP-сервера vsftpd для обмена файлами
- Автоматизация развертывания и управления с помощью Ansible

## Технологии
- Linux (Ubuntu/Debian)
- DNS (dnsmasq), NTP (Chrony), SMTP (Postfix), HTTP (Nginx), FTP (vsftpd)
- Ansible для автоматизации конфигураций
- Systemd, cron, SSH
