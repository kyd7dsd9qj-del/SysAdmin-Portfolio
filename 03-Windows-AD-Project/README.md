# Проект 3: Доменная инфраструктура на Windows Server

## Цель
Развернуть доменную среду Active Directory для централизованного управления пользователями, компьютерами и политиками безопасности в компании.

## Задачи
- Установка ролей Active Directory Domain Services и DNS Server
- Повышение сервера до контроллера домена (practicumsuperstore.local)
- Создание пользователей и организационной единицы (OU) "IT"
- Настройка групповых политик (GPO) для IT-подразделения
- Добавление клиентской машины PCI-1 в домен
- Проверка структуры домена и применения политик

## Технологии
- Windows Server 2022/2019
- Active Directory Domain Services (AD DS)
- DNS (встроенный)
- Group Policy Objects (GPO)
- Организационные единицы (OU)

## Результат
Создана полнофункциональная доменная среда с централизованным управлением учётными записями, компьютерами и групповыми политиками. Доказана работоспособность ключевых компонентов инфраструктуры Windows Server.
